# Obsidian Redirect

Tiny static HTTPS wrapper for opening local Obsidian notes from apps such as Telegram that do not reliably handle `obsidian://` links directly.

Usage:

```text
https://labartw.github.io/obsidian-redirect/?path=/absolute/path/to/note.md
```

or:

```text
https://labartw.github.io/obsidian-redirect/?vault=Vault%20Name&file=Folder%2FNote.md
```

The page stores nothing and has no analytics.
